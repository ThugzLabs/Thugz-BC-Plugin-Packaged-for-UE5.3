# Thugz BC Plugin Packaged
 The packaged plugin of the Thugz Labs BC Plugin for Unreal Engine 

 To use it:
- Download the last version of the release of your engine version
- Create a "Plugins" directory inside your project directory and dezip the plugin dowloaded inside the "Plugins" directory :

<img width="623" alt="image" src="https://github.com/ThugzLabs/Thugz-Solana-for-Unreal-Engine/assets/120092937/c09b4d62-b7f3-416b-a54a-b97a73c089d5">

If your project is a blueprint only, create an empty C++ class to convert it as a C++ projet:

<img width="1613" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/a4abc69f-554b-4f4e-9ff1-0c423234504b">

<img width="852" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/499e9f0a-bdf7-4d75-ab12-a613ae1dd596">

<img width="797" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/ea2c2cfc-50d1-49c9-8964-87c82bd1e36a">

Wait for the code to be added to the project and then, accept the popup:

<img width="415" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/76a84020-e03f-40a0-af23-de9b39719de7">

Then click no:

<img width="475" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/64b56e96-c6d4-447c-a8dc-c45d7333b898">

Then exit the projet and generate Visual Studio file:

<img width="459" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/2d65cc9d-40d5-4062-98b2-f74199adab9a">

Then launch your ptoject. Click Yes here:

<img width="448" alt="image" src="https://github.com/ThugzLabs/Thugz-BC-Plugin-Packaged/assets/120092937/2373d31f-d18a-4ab4-8e96-e7c63c769599">

And wait until your project is loaded (it can take several minutes)

Go to the plugin panel to verify that the Thugz Plugin is correctly activated

<img width="1911" alt="image" src="https://github.com/ThugzLabs/Thugz-Solana-for-Unreal-Engine/assets/120092937/8163eb8d-6532-4384-996c-9d309abed588">


To use our widget test, go to the plugin content:

<img width="478" alt="image" src="https://github.com/ThugzLabs/Thugz-Solana-for-Unreal-Engine/assets/120092937/28d1c694-ec16-40fb-a0b1-ba42fb5f57b8">
